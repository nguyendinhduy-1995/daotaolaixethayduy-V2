# ADMIN_GUIDE_UPDATE_NOTES

## Cập nhật mới cho trang hướng dẫn Admin

- Đã bổ sung nội dung module `Trợ lý công việc` trong `FEATURE_MAP_AND_RUNBOOK.md`.
- Trang `/admin/guide` đọc trực tiếp từ markdown nên tự hiển thị phần mới.
- Mapping quyền UI đã nhận diện thêm:
  - `kpi_targets`
  - `goals`
  - `ai_kpi_coach`

## Hướng dẫn sử dụng nhanh

1. Vào `/ai/kpi-coach` để xem gợi ý AI theo ngày.
2. Dùng nút `Tạo Outbound` cho action phù hợp.
3. Gửi feedback `Hữu ích/Chưa phù hợp` để n8n learning loop cải thiện đề xuất.
4. Vào `/kpi/targets` để chỉnh target theo role/branch/dayOfWeek.
5. Vào `/goals` để nhập mục tiêu ngày/tháng (doanh thu, hồ sơ, chi phí).
